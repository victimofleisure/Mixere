CPU Usage

Mixere typically uses less than 5% of the CPU. However, if you have many documents open, or if your documents contain many tracks, or many automations, CPU usage can increase substantially. If CPU usage becomes a problem, here are some things to try:

  • Reduce the slider display smoothness, using View/Options/Sliders; for details, see Options.
  • Delete empty tracks from your documents.
  • Open less documents at once.
  • Uncheck the "Show window contents while dragging" system property (Settings/Control Panel/Display/Effects); for details, see Resizing Columns.
